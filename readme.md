# Социальная сеть для профессионального сообщества

Дипломный проект. Платформа для общения и обмена опытом специалистов. В разработке.

## Стек
- PHP (MVC-архитектура)
- MySQL
- HTML/CSS

## Статус
В активной разработке. Реализовано:
- Регистрация/авторизация
- CRUD для профилей (личный и группы)
- Базовая структура MVC

Планируется: мессенджер, лента постов, оптимизация запросов.

## Запуск проекта
1. Склонировать репозиторий.
2. Настроить веб-сервер (например, OpenServer) с поддержкой PHP и MySQL.
3. Импортировать дамп БД из папки `database`.
4. Открыть сайт в браузере.
