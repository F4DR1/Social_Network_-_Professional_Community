# Социальная сеть для профессионального сообщества

Дипломный проект. Платформа для общения и обмена опытом специалистов. В разработке.

## Стек
- PHP (MVC-архитектура, без фреймворка)
- MySQL
- HTML/CSS
- JS

## Статус
В активной разработке.

### Реализовано:
- Базовая структура MVC
- Регистрация/авторизация
- CRUD для профилей (личный и группы)
- CRUD для взаимоотношений (подписки)
- CRUD для постов (создание и удаление; вывод в профилях личных и групп)

### Разрабатывается сейчас:
- Лента постов
- Рефакторинг кода (общие методы для избежания повторения кода)
- Подготовка к написанию css (+ перепись старого)

### Планируется в будущем:
- Взаимоотношения (блокировки и списки контактов)
- Мессенджер

## Запуск проекта
1. Склонировать репозиторий.
2. Настроить веб-сервер (например, `OpenServer`) с поддержкой PHP и MySQL.
3. Импортировать дамп БД из папки `database`.
4. Открыть сайт в браузере.
